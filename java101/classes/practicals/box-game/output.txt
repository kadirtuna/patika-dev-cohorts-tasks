Project Name: Box Game

Project Explanation:
This program simulates a boxing match between two fighters using Java classes.
Each fighter has health, damage, weight, and a dodge chance. The Ring class manages the match logic.
As a homework addition, a 50% random chance decides which fighter attacks first each match.
