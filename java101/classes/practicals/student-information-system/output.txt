Project Name: Student Information System

Project Explanation:
This project models a student grade system with Student, Course, and Teacher classes.
Each course supports both exam notes and oral notes, with configurable oral weight percentage affecting the final grade.
The system calculates each student's weighted average and determines if they pass (average > 55).
